What is Hoverme.?


Hoverme.css is a library designed to provide a wide variety of hover and focus effects for your web elements. It is easy to use, lightweight, and customizable.


Whether you are a beginner or an experienced developer, Hoverme.css can help you enhance the interactivity of your website with minimal effort.


You can choose from many HOVER effects that Hoverme offers.

How to install Hoverme.


You need to install Hoverme to use the effects. Here are the ways to install it.


installation

Add it directly to the <head> of your website using a CDN: 

<link rel="stylesheet" 
href="https://cdn-hoverme.netlify.app/hoverme/
1.0.4/hoverme.css">
Learn how to use this library

How to use Hoverme.


This tool is very easy to use, just copy and paste. It's that easy!


The first thing you need to do is install the CSS library. If you haven't already, there are two ways to do this: through a cdn or by directly downloading the Hoverme ZIP file, which you'll find at the top of the page. Click here to download.


Now go to the effects tab, there you will find the entire collection of hover effects, each button has two options, which are to copy the class as hover or focus, once you do this just paste the class into your button, image, logo, etc.


Example:

<a href="/" class="hvr-shrink"></a>
